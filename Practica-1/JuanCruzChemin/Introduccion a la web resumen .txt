Hipertexto: texto con enlaces

 La web es uno de los servicios que funcionan sobre internet y se la considera una interface para utilizar la red

Un protocolo es un conjunto de normas que permite el intercambio de información entre máquinas de diversos tipos conec­tadas entre sí

 FTP (para la transferencia de archivos), POP y SMTP (para el envío y recepción de correo electrónico), TELNET (para la conexión con terminales remotos), GOPHER, WAIS y HTTP (para el acceso a servidores de información). 
El protocolo HTTP (HyperText Transfer Protocol) es la base de la World Wide Web

-Contadores de visitas
-Botones gif
